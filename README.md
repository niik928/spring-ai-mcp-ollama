# Spring AI MCP with Ollama

This project demonstrates a local AI agent built using Spring Boot and Spring AI,
integrated with MCP (Model Context Protocol) and Ollama for open-source LLMs.

The AI can safely create and manage files using MCP filesystem tools.

## Tech Stack
- Java 17
- Spring Boot
- Spring AI
- MCP Client
- Ollama (qwen2.5 / llama3.1)

## Features
- Free & local AI (no OpenAI)
- Secure filesystem access via MCP
- Tool-calling AI agent
- REST API based interaction
